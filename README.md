# 🎓 Campus Connect: Unified Student Onboarding Portal

## 📌 Project Overview
“Campus Connect: Unified Student Onboarding Portal” is software developed for managing various academic and administrative activities within a college through an online platform. It facilitates efficient handling of student information, admissions, scholarships, courses, and facility bookings. The system enables students to apply for admission and scholarships, download ID cards, and view academic details online. The HOD can review applications and manage bookings for facilities like classrooms and auditoriums, while the Scholarship Cell verifies and processes scholarship requests. The Principal has overall control to finalize approvals, manage courses, and monitor all operations. The main objective of this system is to develop a user-friendly platform that simplifies college management, saves time, and enhances coordination between departments.
---

## 🚀 Features
- Student Registration & Login
- Online Admission System
- Academic Details Viewing
- Facility Booking (Classrooms, Auditoriums)
- Scholarship Application & Management
- Admin Dashboard
- Secure Data Management System

## ⚙️ How to Run
1. Install XAMPP/WAMP
2. Place project folder in `htdocs`
3. Start Apache & MySQL
4. Import database
5. Open browser and go to: http://localhost/collegeManagementSystem
